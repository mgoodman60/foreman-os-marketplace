# Construction Daily Report Language Standards

Transform casual user input into professional construction daily report language. These patterns ensure reports are clear, defensible, and consistent.

## Core Principles

1. **Third person, past tense** for completed work
2. **Quantify everything** possible (CY, LF, SF, EA, tons)
3. **Reference locations** (grid lines, building areas, floor levels, rooms)
4. **Use standard terminology** (see glossary below)
5. **Be specific about status** (completed, in progress, ongoing, pending)
6. **Document cause for delays** (weather, material, labor, design, inspection hold)

## Transformation Examples

### Work Performed

| User says | Report says |
|-----------|-------------|
| "Poured concrete today" | "Concrete placement completed for [area]. Approximately [X] CY placed." |
| "Guys were doing rebar" | "Rebar installation in progress at [location]. [X] tons placed to date." |
| "Hung drywall on 2nd floor" | "Drywall installation continued on Level 2, [areas/rooms]. Approximately [X] sheets hung." |
| "Electricians pulling wire" | "[Sub name] continued electrical rough-in, pulling conductors in [area/wing]." |
| "Digging the foundation" | "Foundation excavation in progress. Excavated to [depth] at [grid lines]." |
| "Cleaned up the site" | "General site cleanup and housekeeping performed. Debris removed from [areas]." |
| "Worked on the roof" | "Roofing operations continued. [Material type] installed on [area], approximately [X] squares completed." |
| "Waterproofing the walls" | "Below-grade waterproofing application in progress on [north/south/etc.] foundation walls." |
| "Framing going good" | "Wood/metal framing progressing on schedule at [location]. [X] LF of walls framed." |
| "Setting steel" | "Structural steel erection continued. [X] pieces set at [grid lines/bays]." |

### Weather and Site Conditions

| User says | Report says |
|-----------|-------------|
| "Super muddy" | "Site conditions wet; standing water and soft ground throughout [areas]." |
| "Really hot today" | "Elevated temperatures; heat safety protocols in effect. Water stations maintained." |
| "Rained all morning" | "Intermittent rain throughout morning hours. Exterior work suspended until [time]." |
| "Frozen ground" | "Ground frost present. Subgrade frozen to approximately [X] inches." |
| "Windy, couldn't use the crane" | "High winds exceeded crane operational limits. Crane operations suspended for safety." |
| "Perfect weather" | "Clear skies with moderate temperatures. All operations proceeded without weather impact." |

### Delays and Issues

| User says | Report says |
|-----------|-------------|
| "Concrete truck was late" | "Concrete delivery delayed [X] hours. First truck arrived at [time] vs. scheduled [time]." |
| "Wrong material showed up" | "Incorrect material delivered by [supplier]: received [what arrived] instead of [what was ordered]. PO #[number]. Material refused/accepted pending resolution." |
| "Inspector failed us" | "Inspection by [name/agency] resulted in failure. Deficiencies noted: [list]. Corrective action in progress." |
| "Waiting on RFI" | "Work at [location] on hold pending response to RFI #[number], submitted [date]." |
| "Sub didn't show up" | "[Sub name] not on site today. [X] workers expected, zero present. Superintendent contacted sub foreman." |
| "Had to stop for safety" | "Work stoppage issued at [time] in [area] due to [safety concern]. Operations resumed at [time] after [corrective action]." |

## Grammar Rules

- Capitalize trade names and proper nouns (HVAC, Tyvek, Sonotube)
- Use "approximately" or "approx." when estimating quantities
- Spell out numbers under 10 in narrative; use digits in tables
- Use 12-hour time format with AM/PM
- Dates in MM/DD/YYYY format
- List multiple activities as separate sentences, not run-on
